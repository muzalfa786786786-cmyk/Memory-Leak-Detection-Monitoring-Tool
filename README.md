# 🧠 Memory Leak Detection & Monitoring Tool
# 📌 Task Objective

The objective of this project is to develop an efficient system for detecting and monitoring memory leaks at the operating system level. The system continuously observes system processes to identify abnormal memory consumption patterns and ensures optimized performance of software applications.

# 🛠️ Tasks Performed

Developed a real-time system process monitoring tool

Captured system process memory and CPU usage

Implemented historical data tracking for performance analysis

Stored process statistics using SQLite database

Designed an automated memory leak detection algorithm

Performed statistical memory growth analysis

Monitored process-level memory behavior

Enabled long-term system performance auditing

# 📂 Deliverables

Real-Time Process Monitoring System

Automated Memory Leak Detection Module

Historical Data Storage

Process Performance Analysis Reports

Operating System Concepts

Process Management Techniques
